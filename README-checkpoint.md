# Student Performance Analyzer

**Files**
- main.py : main interactive program
- utils.py : helper functions (totals, grades, algorithms)
- data/sample_students.csv : example dataset
- flowchart.txt : simple flowchart / pseudocode
- course_content.jpg : the course image you provided

**How to run**
1. Ensure you have Python 3 installed.
2. Run `python main.py`
3. Use menu options. Save to CSV to persist any changes.

**Project features mapping to course units**
- Unit 1: Problem solving, top-down design (see flowchart)
- Unit 2: Python basics, functions and modules (utils.py)
- Unit 3: Control flow and fundamental algorithms (kth element, sorting)
- Unit 4: GCD and prime checks included in utilities
- Unit 5: Array techniques, duplicates removal, sorting
- Unit 6: Flowchart provided in flowchart.txt (also use Raptor/Scratch if desired)
